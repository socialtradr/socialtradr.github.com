Why SocialTradr?

{% include why_navbar.html %}

SocialTradr uses data mining algorithms to harvest public opinion from social media networks such as Facebook, Twitter, and Reddit, as well as blogs and online newspapers. This data is sent through a proprietary real-time analytics system to generate reports, calculate insights, and detect trends. Users will be able to access this information through a web-based interface, a Microsoft Excel plugin, and daily or weekly email reports.

SocialTradr is different because we specifically target financial professionals. Our products complement the tools financial professionals already use. We allow customers to track many companies.
